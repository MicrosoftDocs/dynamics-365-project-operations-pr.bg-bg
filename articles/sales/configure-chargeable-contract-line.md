<span data-ttu-id="6fdc3-105">Базираните на проекта аспекти на договор има концепцията за включени компоненти, таксуеми и нетаксуеми компоненти.</span><span class="sxs-lookup"><span data-stu-id="6fdc3-105">A project-based contract line has the concept of included, chargeable, and non-chargeable components.</span></span>

<span data-ttu-id="6fdc3-111">Плащаемостта се определя в категориите трансакция за аспекти на договор по проект и се прилага само за класа трансакция **разход**.</span><span class="sxs-lookup"><span data-stu-id="6fdc3-111">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="6fdc3-112">Ако **Включете разходи** е зададено на **Не** на аспекти на договор по проект, разделът **Платими категории** не е наличен.</span><span class="sxs-lookup"><span data-stu-id="6fdc3-112">If **Include Expenses** is set to **No** on a project contract line, the **Chargeable Categories** tab isn't available.</span></span>
