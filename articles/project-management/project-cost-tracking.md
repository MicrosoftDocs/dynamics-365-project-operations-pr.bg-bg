## <a name="labor-cost-tracking-view"></a><span data-ttu-id="43bbf-108">Изглед за проследяване на разходи за труд</span><span class="sxs-lookup"><span data-stu-id="43bbf-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="43bbf-109">На страницата **Проекти** в раздела **Проследяване**, можете да изберете **Проследяване** > **Разходи**, за да отворите изгледа **Проследяване на разходите**, за да видите напредъка на разходите за труд за всяка задача в плана на проекта.</span><span class="sxs-lookup"><span data-stu-id="43bbf-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="43bbf-110">Тази гледна точка също сравнява действителните разходи за труд, изразходвани за дадена задача, с планираните разходи за труд на задачата.</span><span class="sxs-lookup"><span data-stu-id="43bbf-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="43bbf-111">Project Operations използва следните формули за изчисляване на показателите за разходите за труд:</span><span class="sxs-lookup"><span data-stu-id="43bbf-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>
